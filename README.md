# UTN_PWA_BACKEND

Este proyecto es una maqueta para una app de mensajería similar a slack.

# INSTRUCCIONES:

1. Abrir carpeta con visual studio code.
2. Utilizar el comando Ctrl + Ñ para abrir la terminal.
3. Correr el código "npm i" para instalar las dependencias.
4. Una vez instalado, utilizar el código "npm run dev" para conectar con el servidor.
5. Confirmar la conectividad a partir del mensaje en consola "Conexión con MongoDB exitosa".

# Variables de entorno:

MONGO_DB_URI= Link de servidor de mongo db
MONGO_DB_NAME= Nombre del servidor
GMAIL_PASSWORD= Contraseña de aplicación de gmail
GMAIL_USERNAME= Email
JWT_SECRET_KEY= Key de jsw
URL_FRONTEND= Link del despliegue de frontend
URL_BACKEND= Link del despliegue de backend
